# 赴华旅行补充指南（电话卡与通用注意事项）
## 一、电话卡选择
1. 三种常见类型：
   - 本地运营商漫游（如Maxis漫游）：80多块马币/周，仅提供流量，无本地电话号码，无法打电话，适合仅需上网的用户。
   - 境外流量卡（EC）：需扫描QR码激活，支持使用Facebook、Instagram等境外APP，无本地电话号码，流量套餐价格适中。
   - 香港电信电话卡（推荐）：
     - 费用：电话卡72港币（含100港币话费），流量套餐20港币/2G、98港币/100G、188港币/220G，开通中国号码仅1港币/天。
     - 优势：含中国+香港两个电话号码，支持通话（480分钟通话时长），无需VPN即可使用境外APP，余额永久有效，可复用於香港、澳门旅行。
     - 网速：4G网络，日常使用（看IG、小红书、微信等）无压力，偶尔轻微卡顿。
2. 选择建议：若需要本地电话号码（方便注册APP、紧急联系），优先选择香港电信电话卡；仅需上网可选择境外流量卡或漫游。

## 二、通用注意事项
1. 实名认证：所有支付类、订票类APP均需用护照完成实名认证，建议出发前提前完成，避免抵达后耽误行程。
2. 手续费：绑境外卡支付时，超过200元人民币的消费会产生3%手续费，大额消费可直接刷信用卡，与APP支付手续费一致；通过朋友换汇充值无手续费。
3. 出行准备：
   - 提前注册并完善各APP的个人信息（如乘客信息、支付方式）。
   - 酒店、高铁票提前预订，热门城市旺季房源和车票紧张，避免临时无票/无房。
   - 携带护照原件，作为乘车、住宿、实名认证的核心凭证。
4. 应急联系：若选择带本地电话号码的电话卡，可提前存储中国急救电话（120）、报警电话（110），方便紧急情况使用。
