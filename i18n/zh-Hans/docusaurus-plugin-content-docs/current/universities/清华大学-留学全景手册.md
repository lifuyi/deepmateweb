## 2. 清华大学 (Tsinghua University) - 工科翘楚，创新摇篮

### 学校简介
清华大学成立于1911年，是中国最著名的理工科大学，在工程、计算机科学、建筑等领域享有盛誉。学校位于北京市海淀区，校园美丽，被称为"园林式校园"。

### 留学申请要求
- **本科申请**：高中毕业，HSK四级（180分以上）或相当水平
- **研究生申请**：学士学位，相关学术背景，HSK五级（180分以上）
- **英语授课项目**：托福（80分以上）或雅思（6.0分以上）
- **申请时间**：本科项目通常在9月至11月开放申请

### 专业推荐
1. **计算机科学与技术**：国内顶尖，国际知名
2. **建筑学**：历史悠久，实力雄厚
3. **机械工程**：传统优势学科
4. **电子工程**：实力强劲，就业前景好
5. **经济管理**：理工背景商科，交叉学科优势

### 就业前景
清华毕业生就业率超过96%，主要去向：
- 高科技企业（华为、腾讯、百度等）
- 金融投资行业
- 制造业龙头企业
- 创业企业
- 国内外深造

### 留学费用
- **学费**：工科约10,000-15,000元/年，经管类约12,000-18,000元/年
- **住宿费**：学生公寓约1,500-2,500元/年
- **生活费**：北京约1,500-2,500元/月

### 奖学金机会
- 中国政府奖学金（CSC）
- 清华大学奖学金
- 企业赞助奖学金
- 国际学生学费减免

### 北京生活指南
- 清华大学位于北京海淀区，高校集中
- 周边有清华园、圆明园等景点
- 北京冬季寒冷，需准备充足冬装
- 交通便利，地铁网络发达
