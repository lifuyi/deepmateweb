## 山东大学 (Shandong University, SDU) - 齐鲁大地，学科齐全

### 学校简介
山东大学是一所历史悠久、学科齐全的综合性研究型大学，在数学、医学、文学等领域具有较强实力。

### 留学申请要求
- **本科申请**：高中毕业，HSK四级（180分以上）
- **研究生申请**：学士学位，相关学术背景，HSK五级（180分以上）
- **英语授课项目**：托福（80分以上）或雅思（6.0分以上）
- **申请时间**：本科项目通常在9月至11月开放申请

### 专业推荐
1. **数学**：国内顶尖
2. **医学**：传统优势
3. **中国语言文学**：实力雄厚
4. **历史学**：发展迅速
5. **经济学**：特色鲜明

### 就业前景
山大毕业生就业率约93%，主要流向：
- 医疗健康产业
- 教育机构
- 政府机构
- 企业
- 创业领域

### 留学费用
- **学费**：文科约8,000-12,000元/年，医科约10,000-15,000元/年
- **住宿费**：学生公寓约1,200-2,000元/年
- **生活费**：济南约1,200-2,000元/月

### 奖学金机会
- 中国政府奖学金（CSC）
- 山东省外国留学生奖学金
- 山东大学专项奖学金
- 企业赞助奖学金

### 济南生活指南
- 济南是山东省会，泉城
- 山东大学位于济南，环境优美
- 济南气候宜人，四季分明
- 济南生活成本相对北京、上海较低
