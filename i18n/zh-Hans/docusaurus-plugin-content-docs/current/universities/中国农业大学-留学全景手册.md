## 36. 中国农业大学 (China Agricultural University, CAU) - 农业科学，国之根本

### 学校简介
中国农业大学成立于1905年，位于北京，是以农业科学为优势和特色的综合性研究型大学，在农业、生命科学、食品等领域具有较强实力。

### 留学申请要求
- **本科申请**：高中毕业，HSK四级（180分以上）
- **研究生申请**：学士学位，相关学术背景，HSK五级（180分以上）
- **英语授课项目**：托福（80分以上）或雅思（6.0分以上）
- **申请时间**：本科项目通常在9月至11月开放申请

### 专业推荐
1. **农业技术**：国内顶尖
2. **植物保护**：传统优势
3. **动物科学**：实力雄厚
4. **食品科学与工程**：发展迅速
5. **生物技术**：特色鲜明

### 就业前景
农大毕业生就业率约92%，主要流向：
- 农业企业
- 食品企业
- 政府机构
- 科研机构
- 国内外深造

### 留学费用
- **学费**：农科约8,000-12,000元/年，生物约10,000-15,000元/年
- **住宿费**：学生公寓约1,200-2,000元/年
- **生活费**：北京约1,500-2,500元/月

### 奖学金机会
- 中国政府奖学金（CSC）
- 北京市外国留学生奖学金
- 中国农业大学专项奖学金
- 企业赞助奖学金

### 北京生活指南
- 中国农业大学位于北京，海淀区
- 北京作为首都，资源丰富
- 北京冬季较冷，需准备保暖衣物
- 可利用周末游览北京历史文化景点
