# 订房订票类APP使用指南
## 一、Trip.com（携程海外版）
1. 核心功能：酒店预订、高铁票购买，专为外国人赴华设计，适配境外用户需求。
2. 酒店预订要点：
   - 筛选功能：支持按城市（如上海）、入住日期（如7月1日-7月10日）、评分（星星等级）、评价数量筛选，部分酒店含6500+点评。
   - 评价查看：可按“最新”“评分从低到高”排序，重点关注负面评价，区分“酒店问题”（如无窗、卫生差）和“客人个人问题”（如操作不当）。
   - 关键选择：需确认酒店“是否有窗”（有窗房型略贵，空气流通性好），结合自身需求决策。
3. 高铁票购买要点：
   - 站点选择：明确出发/到达城市的具体火车站（如上海虹桥站、南站等），避免选错。
   - 座位类型：二等座（3+2布局，约22元，性价比高）、一等座（2+2布局，更宽敞）、商务座（独立座位，约72元），按需选择。
   - 提前准备：添加乘客信息（外国人需填写护照号），避免当天购票耽误时间。
4. 优势与福利：提供推荐链接，注册后预订可享折扣或福利（非强制使用），酒店资源覆盖广，支持外国人预订。

## 二、携程（国内版）
1. 核心功能：酒店预订，与Trip.com同源，资源互通。
2. 核心优势：以国内用户评价为主，反映酒店本地真实口碑，可与Trip.com的外文评价交叉验证。
3. 使用场景：搭配Trip.com使用，全面了解酒店情况后再决策，避免单一评价来源的局限性。

## 三、中国铁路12306（官方高铁订票APP）
1. 核心功能：高铁票购买、改签，官方渠道无额外服务费。
2. 注册与认证：
   - 语言设置：需切换为中文版本（英文版本可能白屏或功能缺失）。
   - 信息填写：用护照号注册，完成实名认证（上传护照照片、本人手持护照照片）。
   - 邮箱核验：注册后查看邮箱（含垃圾邮箱），完成邮件核验方可正常使用。
3. 购票规则：
   - 开票时间：高铁票提前2周开票，可设置开票提醒，避免错过购票时间。
   - 乘客添加：最多支持添加9名乘客，需填写每位乘客的护照信息。
   - 座位选择：含二等座、一等座、商务座、无座，多人出行可选择连座。
4. 改签与乘车：
   - 改签：赶不上高铁可在APP内改签，差价多退少补，无额外手续费。
   - 乘车凭证：需携带护照，出发前30分钟开始检票，外国人需走人工通道。
5. 注意事项：境外网络支付可能加载失败，遇问题可切换至Trip.com购票。
