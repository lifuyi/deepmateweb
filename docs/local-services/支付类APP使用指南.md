# 支付类APP使用指南
## 一、支付宝
1. 核心功能：日常支付、交通出行（地铁/公交/打车）、转账收款，覆盖境内绝大多数消费场景。
2. 注册与认证：
   - 语言设置：必须切换为中文版本（中英版本功能差异大）。
   - 注册方式：支持用微信号注册，外国人需用护照完成实名认证（填写职业、地址，部分场景需人脸识别）。
3. 充值方式：
   - 私下换汇（推荐）：找信任的中国朋友换汇（转马币给朋友，朋友转人民币到支付宝），无手续费。
   - 绑卡充值：支持境外信用卡/借记卡（Visa、Master等），消费超过200元人民币征收3%手续费。
4. 支付场景：
   - 日常消费：扫码支付（店家提供QR码）或出示个人QR码让店家扫码，支付顺序可设置（优先余额、信用卡等）。
   - 交通出行：
     - 地铁/公交卡：切换至出行城市，在“出行”板块开通地铁卡、公交卡（需同意协议、设置6位支付密码），生成专属二维码，进出站/上下车直接扫码扣费。
     - 滴滴打车：无需额外下载APP，在支付宝内使用，支持定位上下车点、选择车型（推荐“一口价”，避免拼车），可拍周边环境照片发给司机对接。
5. 注意事项：地铁卡和公交卡需分别开通，不可通用；大额消费（1000元以上）可直接刷信用卡，与APP支付手续费一致。

## 二、微信（含微信支付、小程序）
1. 核心功能：支付结算、小程序点餐/团购、社交沟通，与支付宝互补覆盖消费场景。
2. 微信支付设置：
   - 地区切换：需将微信地区切换至“中国大陆”（否则可能显示服务不可用）。
   - 实名认证：绑定银行卡或通过余额充值完成实名认证，需上传护照照片验证。
3. 支付场景：
   - 日常支付：支持扫码支付，与支付宝功能类似，可作为备用支付方式。
   - 小程序点餐：80%以上餐馆支持微信扫码小程序点餐（如霸王茶姬、阿妈手作等），可提前点餐减少等待时间，仅支持微信支付。
4. 注意事项：
   - 小程序点餐需授权地理位置，选择就近门店，支付仅支持微信余额或绑定的银行卡。
   - 绑境外卡消费超过200元人民币征收3%手续费，建议通过余额支付避免额外费用。
